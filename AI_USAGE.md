# AI Usage

This project made limited and responsible use of AI-assisted tools as learning and support aids.

## ChatGPT
ChatGPT was used as an interactive assistant to help with:
- Debugging and understanding Python code
- Clarifying machine learning and data science concepts
- Improving code structure and readability
- Refining the wording and coherence of the written report

## GitHub Copilot
GitHub Copilot was used within the development environment to assist with:
- Code autocompletion and boilerplate generation
- Suggesting syntax and function usage during implementation
